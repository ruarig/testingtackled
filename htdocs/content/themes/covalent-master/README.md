# covalent

Tema para Ghost, dando protagonismo a las imágenes que acompañan cada post. Demo disponible en [www.gersonlazaro.com](http://www.gersonlazaro.com).

# Instalación

* Descomprimir o clonar en la carpeta content/themes de tu instalación de Ghost.
* En el archivo `post.hbs` es necesario editar la linea 164 reemplazando YOUR_USERNAME por tu nombre de usuario en disqus (esto es requerido para insertar comentarios en las publicaciones). Si no tienes cuenta en disque puedes crear una gratuita [aquí](http://www.disqus.com)
* Reinicia tu servicio de ghost.
* En el dashboard de administrador de Ghost sección "settings" abre "general" y en la selección "theme" elige covalent.

# Compatibilidad

Este tema ha sido testeado en Ghost 0.8.0 (actual). No se garantiza compatibilidad con versiones anteriores, aunque es muy posible que funcione sin problemas (bienvenidas sean las pruebas que quieran realizar).

# Optimización de imágenes

Dado que el tema da prioridad en el inicio a las imágenes que acompañan cada post, es recomendable reducir el tamaño de las imagenes de portada de tus publicaciones. Para esto puedes usar [Compressor.io](https://compressor.io/) que reduce el peso de las imagenes sin perder calidad.
